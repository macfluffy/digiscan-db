# Dependencies
Express - Web framework https://expressjs.com/
Sequelize - A node.js ORM tool that is compatible with Postgres. Requires Node v10+ https://sequelize.org/docs/v6/
jsonwebtoken - Create tokens to authorize user access https://github.com/auth0/node-jsonwebtoken#readme
pg - https://github.com/brianc/node-postgres
pg-hstore - Converts json into Hstore format https://github.com/scarney81/pg-hstore
dotenv - Loading environment files from the host's .env files https://github.com/motdotla/dotenv#readme

Middlewares: These are all security features
Cors - Tells browsers the origins of the request/responses https://github.com/expressjs/cors#readme
Helmet - Secures express apps by setting HTTP headers https://helmetjs.github.io/
Express-Rate-Limit - Limit the number of repeated requests to the API https://github.com/express-rate-limit/express-rate-limit
